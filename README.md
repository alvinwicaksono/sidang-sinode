sidang sinode gkj

Langkah Install

1. Copy file .env.example dan paste di tempat yang sama
2. Lalu rename menjadi .env dan setting database di dalam .env
3. composer install
4. npm install
5. npm run dev
6. php artisan migrate:refresh --seed
7. php artisan key:generate
8. php artisan storage:link
9. php artisan serve
